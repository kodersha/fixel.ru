Команды бота

!время
Время длительности стрима
!совет
Спросить совет у бота
!обнять
Обнять кого-то из чата
!любовь
Узнать процент любви с кем-то из чата

Баллы канала

Каждые пять минут стрима зрителям начисляется по 25 баллов канала которые можно использовать для различных команд чат-бота.

!баллы
Узнать своё количество баллов
!первый
Получить бонусные баллы придя на стрим первым

Озвучивание сообщений ботом

!сказать200 баллов канала
Озвучить сообщение ботом 
Пример: !сказать текст сообщения

Битва

Ставка на битву составляет 600 баллов канала, победитель получает 600 баллов проигравшего.

!битва600 баллов канала
Вызвать участника чата на битву 
Пример: !битва meowcat
