Баллы канала

За каждое сообщение зрителям начисляется по 5 баллов канала которые можно использовать для различных команд чат-бота.

!баллы
Узнать своё количество баллов

Озвучивание сообщений ботом

!сказать25 баллов канала
Озвучить сообщение ботом 
Пример: !сказать текст сообщения
