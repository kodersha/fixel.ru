Streamlabs Chatbot · Озвучивание выделенного сообщения на Twitch

Для Streamlabs Chatbot существует скрипт TheNewTTS, он позволяет боту озвучивать сообщения из чата твича голосами Google. С появлением баллов канала, некоторые стримеры обзавелись возможностью озвучивания выделенных сообщений, чтобы иметь такую возможность у себя, я слегка дописала плагин TheNewTTS и опубликовала его на GitHub. Теперь, если в настройках плагина не включена функция «Озвучивать всё», он будет озвучивать только те самые, выделенные за баллы канала, сообщения чата.

Скачать скрипт. Не забудьте переименовать папку в ThePointsTTS после распаковки архива. В настройках скрипта, во вкладке TTS Configuration, выберете русский язык озвучки.

Озвучивание голосом IVONA Maxim

TTS Alerts and Chat — похожий на предыдущий скрипт, но с поддержкой голоса IVONA Maxim. Имеет возможность озвучивания сообщений за баллы канала, по команде чата и ещё огромное количество разнообразных настроек, в том числе для юутуба и миксера.

Для использования голоса IVONA придется купить (Или скачать) отдельную программу Speech2Go. В настройках самого скрипта, в разделе TTS Command, можно указать количество баллов, которое будет использоваться для озвучивания сообщений. Например, если указать 250 баллов и создать в настройках канала отдельную награду, стоимостью 250 баллов, скрипт станет озвучивать сообщения отправленные именно из этой награды.

Инструкция по установке скриптов

Установите Python 2.7.13, если не ставили его при установке Streamlabs Chatbot. Скачайте один из скриптов для озвучивания сообщений. Распакуйте архив и скопируйте его в папку Scripts вашего чат-бота. Обычно, если вы не меняли стандартную папку установки чат-бота, она находится по пути:

C:\Program Files\Streamlabs Chatbot\Services\Scripts

Запустите Streamlabs Chatbot и откройте вкладку Scripts, нажмите на шестерёнку в правом верхнем углу окна.

Укажите путь до папки Lib установленного Python 2.7.13 и перезапустите Streamlabs Chatbot. По умолчанию он устанавливается в корень диска C.

C:\Python27\Lib

Также есть англоязычное наглядное видео, о том как устанавливать скрипты в Streamlabs Chatbot. Ещё можно почитать о настройке и установке самого чат-бота.
